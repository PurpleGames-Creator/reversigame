# リバーシオンライン対戦ゲーム

スマートフォン最適化のリアルタイムオンライン対戦リバーシゲーム。

## セットアップ

### サーバーの準備

```bash
cd server
npm install
```

### クライアントの準備

```bash
cd client
npm install
```

## 開発起動

### ターミナル1（サーバー）

```bash
cd server
npm run dev
```

Expected: `Server running on http://localhost:3001`

### ターミナル2（クライアント）

```bash
cd client
npm run dev
```

Expected: `ready - started server on 0.0.0.0:3000`

ブラウザで http://localhost:3000 にアクセス。

## 技術スタック

- **フロントエンド:** Next.js 14, React 18, Tailwind CSS 3, Socket.io Client
- **バックエンド:** Node.js, Express 4, Socket.io 4, CORS
- **プロトコル:** WebSocket (Socket.io)
- **スタイリング:** Tailwind CSS (レスポンシブ・モバイル最適化)

## ゲームルール

- **盤面:** 8×8 のリバーシ盤
- **制限時間:** 1手 20秒
- **タイムアウト:** 時間切れは自動的にランダムな合法手を配置
- **終局:** 両者置くところがない状態で終了
- **勝敗:** コマ数の多い方が勝利（同数は引き分け）
- **基本ルール:** 敵コマを挟んだら裏返る（リバーシ / オセロ）

## ゲーム画面

### タイトル画面
- プレイヤー名入力
- 待機中の人数表示（リアルタイム）
- マッチング開始ボタン

### ロビー画面
- **募集中:** 新規ゲーム募集中の部屋一覧
- **対戦中:** 現在進行中の対戦表示
- 新規ルーム作成ボタン
- 手動更新ボタン

### ゲーム画面
- 8×8 リバーシ盤面（白・紫のコマ）
- プレイヤー名とコマ数表示
- 20秒カウントダウンタイマー
- 合法手マーカー（白い丸）
- 直前手マーカー（赤い丸）
- 降参ボタン / 退室ボタン

## デザイン特性

- **モバイル最適化:** iPhone SE ～ iPhone 14 Pro Max 対応
- **プレミアムUI:** グラデーション・シャドウで高級感を演出
- **アニメーション:** コマ配置時はフェードイン、直前手は回転演出
- **レスポンシブ:** ダブルタップズーム無効、タッチ操作最適化

## ビルド・デプロイ

### Next.js ビルド

```bash
cd client
npm run build
npm run start
```

### Docker での実行

```bash
docker build -t reversi-game .
docker run -p 3000:3000 -p 3001:3001 reversi-game
```

http://localhost:3000 でアクセス可能

## トラブルシューティング

- **サーバーに接続できない:** サーバーが 3001 ポートで起動しているか確認
- **ゲーム画面が表示されない:** ブラウザコンソールでエラーを確認
- **モバイルでズームされる:** キャッシュをクリアして再度アクセス
